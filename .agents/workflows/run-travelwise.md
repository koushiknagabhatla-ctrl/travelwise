---
description: Start the TravelWise Frontend and Backend Gateway
---

This workflow starts both the frontend and backend services concurrently.

1. Navigate to the frontend directory:
   `cd frontend`

// turbo
2. Start the development environment:
   `npm run dev:all`

3. The website will be available at:
   - Frontend: http://localhost:5173
   - Backend: http://localhost:5002
